# Jaaz's Portfolio

## Overview
This repository contains the source code for my personal portfolio website. It showcases my skills, projects, and professional experience in IT and web development.

## Technologies Used
- HTML5
- CSS3 (Tailwind CSS)
- JavaScript (React)

## Features
- Responsive design
- Project showcase
- Skills and expertise highlights
- Contact information
- About me section

## Setup
1. Clone the repository
2. Open `index.html` in your browser

## Customization
This site was built using the "Spotlight" Tailwind CSS template as a starting point, then heavily modified to suit my personal style and needs.
